# Sonata
Welcome to this new python VSRG

## How to run the python file
Install [Python 3.11.9](https://www.python.org/downloads/release/python-3119/)

Open terminal in project folder

Create a fresh new python environment for the project with this command:
```bat
python -m venv sonata-venv
sonata-venv\Scripts\activate
```
If on Linux, run this instead:
```sh
python -m venv sonata-venv
source sonata-venv/bin/activate
```

Install necessary modules:
```bat
python -m pip install requests pygame pyyaml
```

Now run:
```bat
python sonatabackend.py
```

Have fun
